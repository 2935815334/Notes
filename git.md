2.命令

用户信息配置

```
$ git config --global user.name "2935815334"
$ git config --global user.email 2935815334@qq.com
```

代码克隆

```
$ git clone
```

提交文件

```
$ git init
$ git add .
$ git commit -m "备注"
```

文件恢复

```
$ git checkout HEAD 文件名及后缀
```

查看提交历史

```
$ git log
```
